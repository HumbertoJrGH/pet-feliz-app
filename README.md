# Pet feliz

O presente projeto, desenvolvido para a disciplina de Desenvolvimento de Aplicações Móveis, tem como objetivo compor a entrega do trabalho final. O mesmo se refere à uma aplicação mobile utilizando React Native, bibliotecas populares e o ambiente de execução Expo.

# Funcionalidades
- Autenticação
- Cadastro de usuários
- Recuperação de senha
- Visualização do perfil do usuário
- Página inicial
- Catálogo de produtos
- Catálogo de serviços
- Pagamento por produtos / serviços
- Contato com funcionários através de chat
- Feed de eventos e campanhas.

# Tecnologias envolvidas
- React Native
- Expo

# Instalação

1. Clone o repositório:

```bash
https://github.com/HumbertoJrGH/pet-feliz-app.git
```

2. Instalação de dependências:

```bash
npm intall
```

3. Execução do projeto:

```bash
npm expo start
```
